# Java MySQL Booking System

A simple console-based Java application integrated with MySQL using JDBC.

---

## 🛠 Tech Stack
- Java
- MySQL
- JDBC

---

## 📂 Project Structure
```
src/
 ├── DBConnection.java
 ├── Main.java
```

---

## ⚙️ Setup Instructions

1. Install MySQL
2. Create a database:
   ```sql
   CREATE DATABASE busbooking;
   ```
3. Update DB credentials inside `DBConnection.java`
4. Run the project

---

## 🚀 Features (Day 1)
- Database connectivity using JDBC
- Basic project setup
